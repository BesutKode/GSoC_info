The purpose of this project is to make the GCJ (GNU Compiler for Java , also called GJC: GNU Java Compiler) work with RTEMS. First of this project is porting GCJ run-time to RTEMS, then run the GCJ Test Suite for the porting, and last submit modifications to the related projects such as GCC, RTEMS and so on. After completing this project, we can use Java for programming RTEMS applications.
