Although there has been many projects within the OSM community that already provide print support, notably Maposmatic and townguide, there is still a lot of ground to get creative given OSM open source nature. In this project, it is aimed to create a web application where the user can customize the way that she would like her maps to be printed for her use. Also, one of the objectives of this project is to keep an eye on mapnik new features proposed for this GSoC related to this same issue.
