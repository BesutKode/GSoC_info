Metalink/HTTP as mentioned in the RFC 6249 provides the facility of providing better download experience same as that of Metalink/XML. It does so by using standards such as ETFlags, Instance Digests in HTTP and Web Linking. On the other hand KGet is a versatile and user friendly download manager of the KDE desktop envirnoment with existing support for Metalink/XML. This project intends to extends the capabality of KGet by extending its support to Metalink/HTTP.
