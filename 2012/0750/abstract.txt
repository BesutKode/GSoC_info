Apache Wookie is a Java server application in the incubation status at ASF. It allows administrators to upload and deploy Widgets packaged according to the W3C Widgets specification. These widgets are then hosted in container applications which connects with Wookie server and let it users to pick widgets to be added to their pages. This GSoC project aims to implement the W3C XML Digital Signatures for Widgets specification, in Wookie. With this feature, organizations can automate the installation and updating process of widgets, verifying the signature.xml of the widget to be deployed or updated.
