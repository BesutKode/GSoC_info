OGSA-DAI is a well know middleware that has the objective of exposing and integrating databases in the grid offering a client API for accessing the exposed databases. It was developed an JDBC Driver for OGSA-DAI aiming to bring legacy applications to the grid and also provides Java developers with a well know method to access relational databases in the grid. This project aim to add new features to the actual version of the JDBC Driver for OGSA-DAI delivering at the end of GSoC the version 0.2.
