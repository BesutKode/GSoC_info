Increase the efficiency of decision making in Sahana where users need to view data in abstract views. This data is from a single Sahana instance or from multiple Sahana instances. In current system there is no user driven mechanism to  integrate this data sources and generate user view according to there requirements. The Aggregator Mashup Portal facilitates user to generate customize views by drag and drop data items he required by using web services and RSS that each Sahana instance provide.
