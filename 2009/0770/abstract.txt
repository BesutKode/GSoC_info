Apache Rampart is the security module of Axis2 which implements the specifications of WS-Security stack. The objective of this Google Summer of Code project is to improve the tests of Rampart in such a way that it covers all possible scenarios including negative scenarios and the feature additions which do not have tests at the moment. Tests for Binding level policy configuration, code generated stubs ,secured MTOM and for negative scenarios will be implemented under this project.
