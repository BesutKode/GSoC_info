RealTrack is an Android application designed to facilitate day-to-day data collection by Peace Corps Volunteers in the field. The main aim of the project is to deploy the current features of RealTrack in iOS including adding/deleting/updating Projects,Assignments and Participation.I also intend to add report generation and create graphs  using these data points.Secondly, I plan to add new features in the android app(mentioned below).I also intend to create line graphs for the participation data.
