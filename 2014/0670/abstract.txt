Currently in pysoy the games are playable using the the client. But, the problem is that each time the user has to find the link to the game in order to play it. If all the games are cataloged in a single place this problem won't arise. But a game may be found anywhere in the internet. So, it would be convenient to the users if they can bookmark their favorite games in their accounts. Next time wherever they may be, by simply logging in users can play their favorite games.
